# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of Lighter SDK
- Complete TypeScript implementation of Lighter API
- Root API endpoints (status, info)
- Account API endpoints (account management, PnL, liquidations, etc.)
- Order API endpoints (order books, active/inactive orders, trades, stats)
- Transaction API endpoints (send transactions, history, logs)
- Market API endpoints (candlesticks, funding rates, bridge info)
- Explorer API endpoints (blocks, batches, markets, search, stats)
- SignerClient for authenticated operations with EIP-712 transaction signing
- Comprehensive TypeScript type definitions
- HTTP client with automatic retry logic
- Error handling with detailed error information
- Full unit test coverage
- Integration tests with mock server
- Complete documentation and examples
- Support for mainnet and staging networks
